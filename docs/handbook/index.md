# Handbook

## What it is

A learning curriculum for building a game engine in C++, written alongside
this engine. Each track is a short sequence of pages: one concept per page,
under 900 words, a runnable code block where one applies, and a sources list
at the bottom. Read the tracks in order — later ones assume the earlier ones.

## Why you care

Engine development material is scattered across decade-old forum threads,
excellent-but-dense books, and videos of wildly varying accuracy. This
handbook is the path actually walked to build the engine documented on this
site, with the dead ends removed.

!!! info "How pages are verified"
    Every page is fact-checked against primary sources before publishing, and
    every code block is compiled and run as pasted. The process reduces
    errors; it cannot guarantee zero. Found one?
    [File an issue](https://github.com/IsItJeff/game-engine/issues).

## What to expect

Pages follow a fixed template — *What it is / Why you care / Quick start /
How it works / Pros & cons / What to expect / Go deeper* — so you always know
where to look. Diagrams over prose for any mechanism longer than two
sentences. Videos are linked with a length and a "watch after reading" note,
never embedded.
